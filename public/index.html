FreeCodeCamp Project: API: Image Search Abstraction Layer

Example usage:

https://img-search-ab.herokuapp.com/api/imagesearch/{search term}?offset={display_num}

Return an array of {display_num} json formatted string:

{"url":"http://data.whicdn.com/images/30855761/large.jpg", "snippet":"cat and funny image", "thumbnail":"https://encrypted-tbn2.gstatic.com/images?q=tbn:ANd9GcRcVvn_L7D1b54fIv-1UZH1V_k1SJPRtamtdlJSjldoolzYEYZYj_XKtjzD", "context":"http://weheartit.com/entry/30855761"}

https://img-search-ab.herokuapp.com/api/latest/imagesearch/

Return:

{"term":"latest search","time":"2016-08-02T02:13:55.828Z"}
